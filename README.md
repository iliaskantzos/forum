# Forum Web Application

Web εφαρμογή forum με δυνατότητα εγγραφής χρηστών, δημιουργίας θεμάτων και απαντήσεων.

## Λειτουργίες
- Εγγραφή & Login χρηστών
- Δημιουργία νέων threads
- Απαντήσεις σε threads
- Προφίλ χρήστη
- Reset password
- Email verification
- Admin panel για διαχείριση

## Τεχνολογίες
- PHP
- MySQL
- HTML / CSS / Bootstrap
- JavaScript

## Βάση Δεδομένων
Για να τρέξει η εφαρμογή:
1. Δημιουργήστε database
2. Κάντε import το αρχείο `forum.sql`
